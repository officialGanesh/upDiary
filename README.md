# upDiary : be creative 

upDiary is a blog app created using reactjs and firebase

## Tech Stack

**Client:** React,TailwindCSS

**Service:** Firebase

## Run Locally

Clone the project

```bash
  git clone git@github.com:officialGanesh/upDiary.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start / yarn start
```


## Features

- dark mode 
- Google auth
- upload blog
- delete blog
- user profile
